
1. Подключаем два коммуттора двумя пересеченными кабелями
2. заходим в настройки коммутаторов и вводим:
## rstp protocol settings

	```
	enable
	configure t
	spanning-tree mode rapid-pvst
	```

## Статическое Агрегирование каналов
3. Подключаем их портами fastEthernet 0/1 и fastEthernet 0/2

```
enable
configure t
interface range fastEthernet 0/1-2
channel-group 1 mode on
write memory
```

## Динамическое Агрегирование каналов

3. Подключаем коммутаторы 2-го уровня к коммутатору 3-го уровня
4. Заходим в настройки главного коммутатора 3-го уровня:
```
enable
conf t
interface range fastEthernet 0/1-2  - порты для подключения к первому коммутатору
channel-protocol lacp
channel-group 1 mode active   - 1 mode это первая группа агрегированных портов
```
5. Далее соединяем другие коммутаторы
6. Настраиваем остальные коммутаторы:
```
enable
conf t
interface range fastEthernet 0/1-2  
channel-protocol lacp
channel-group 1 mode passive
```
