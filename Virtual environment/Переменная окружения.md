```
export PATH=$PATH:/home/ular/apps/. 
```

```
Вот что делает эта команда:

1. `export PATH`: Экспортирует переменную окружения `PATH`, чтобы она была доступна для всех подпроцессов, запущенных в текущей оболочке.
    
2. `$PATH`: Использует текущее значение переменной `PATH`.
    
3. `:/home/ular/apps/.`: Добавляет к текущему значению `PATH` новый путь `/home/ular/apps/.`, разделяя его двоеточием (`:`).
    

Таким образом, после выполнения этой команды, система будет также искать исполняемые файлы в директории `/home/ular/apps/.` при запуске команд.
```
